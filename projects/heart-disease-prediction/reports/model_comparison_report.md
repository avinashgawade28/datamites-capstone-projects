# Model Comparison Report: Heart Disease Prediction

## 1. Introduction
This report evaluates several machine learning models to predict heart disease in patients based on clinical features.

## 2. Models Evaluated
- **Random Forest**: An ensemble of decision trees.
- **Logistic Regression**: A linear model for binary classification.
- **Support Vector Machine (SVM)**: A robust classifier for small datasets.

## 3. Performance Summary
| Model | Accuracy | F1-Score | Strength |
|-------|----------|----------|----------|
| Random Forest | ~83.3% | 0.80 | Handles non-linear relationships well. |
| Logistic Regression | ~80.6% | 0.77 | Highly interpretable. |
| SVM | ~80.6% | 0.77 | Effective in high-dimensional spaces. |

## 4. Conclusion
The **Random Forest** model is recommended for its slightly superior accuracy and ability to rank feature importance, which is valuable for clinical insights.
